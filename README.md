# arcade-box
A plan for a arcade box for two players


# Electronic components
## Arcade components
- Sanwa JLF-TP-8YT joystick
  - 2x
  - http://www.arcadeshop.de/Sanwa-Joysticks-JLF-TP-8YT-Sanwa_627.html
- Sanwa OBSF-30 buttons
